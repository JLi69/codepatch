# CodePatch

This is a game that I made for a University game jam during spring break in
2026. I started three days late so I only had about four days to make something.
This is what I came up with.

The theme was "patchwork" and I one of my thoughts when I heard that was
"*work*ing on *patch*ing bugs" in software development. Hence, I made a game that
was themed around that.

In this game you run around a procedurally generated maze collecting 3 patch files 
while avoiding bug enemies. Once you collect all 3 patch files you have to survive 
a wave of enemies and then you can go to the next level.

Overall, I think this is actually a pretty fun game and I'm happy with how it
turned out. You can play the game on [itch.io](https://nullptr-error.itch.io/codepatch). 
Enjoy!

You can find credit for third party assets/artwork in `assets/credits.txt`.
